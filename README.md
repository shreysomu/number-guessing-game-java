# 🎯 Number Guessing Game in Java

A simple console-based Number Guessing Game developed using Java. The program generates a random number between 1 and 100, and the player must guess the correct number within a limited number of attempts.

This project demonstrates fundamental Java concepts such as loops, conditional statements, methods, user input handling, and random number generation.

---

## 🚀 Features

* Generates a random number between 1 and 100
* Allows the user to guess the number
* Provides hints if the guess is too high or too low
* Limited number of attempts
* Displays a success message when the correct number is guessed
* Reveals the correct number if all attempts are exhausted

## ▶️ How to Run

### Clone the Repository

```bash
git clone https://github.com/shreysomu/number-guessing-game-java.git
```

### Navigate to the Project

```bash
cd number-guessing-game-java
```

### Compile the Program

```bash
javac NumberGuessingGameInJava.java
```

### Run the Program

```bash
java NumberGuessingGameInJava
```
